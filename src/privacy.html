iGravity Studios, LLC built Matchimals.fun as an Open Source application. Matchimals.fun is provided at no cost and is intended for use as-is.

Matchimals.fun contains no advertisements and collects no personal information. There are no analytics or third-party tracking services involved- the application is 100% offline and does not require a connection to the internet.

All application source code is available at https://github.com/igravitystudios/matchimals.fun
