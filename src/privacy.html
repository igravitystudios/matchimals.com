iGravity Studios, LLC built Matchimals.fun as an Open Source application. Matchimals.fun is provided at no cost and is intended for use as-is.

Matchimals.fun contains no advertisements and collects no personal information. The application does not require a connection to the internet.

Matchimals.fun uses Firebase Analytics to collect anonymous usage data. This helps us understand how users interact with our app so we can improve the game. We do not collect any personal information through this process.

All application source code in it's entirety is available at https://github.com/igravitystudios/matchimals.fun
